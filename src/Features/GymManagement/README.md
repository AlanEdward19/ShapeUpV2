# GymManagement

Domain focused on gym operations, trainer operations, platform tiers, and role composition.

Main capabilities:
- Platform tier CRUD (ShapeUp subscription tiers)
- Multi-role user assignment (`Trainer`, `Client`, `GymOwner`) with optional platform tier link
- Gym lifecycle and gym-level plan CRUD
- Gym staff management (`Trainer`, `Receptionist`)
- Gym client enrollment and trainer assignment/transfer inside gym context
- Independent trainer plan/client management for non-gym trainers

See `ARCHITECTURE.md` for full flow and schema details.
